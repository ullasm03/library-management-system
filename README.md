# Library Management System

## 📌 Project Description
This is a web-based Library Management System developed using Java, Spring MVC, Hibernate, and JSP. It helps manage books, students, and book issuing operations.

## 🚀 Features
- Add and manage books
- Register and manage students
- Issue and return books
- Track issued book records

## 🛠️ Technologies Used
- Java
- Spring MVC
- Hibernate (JPA)
- JSP
- Maven

## 🏗️ Architecture
- Controller Layer
- Service Layer
- DAO Layer
- Database Layer

## 📂 Project Structure
- src/main/java → Java source code
- src/main/webapp → JSP pages
- pom.xml → Dependencies
